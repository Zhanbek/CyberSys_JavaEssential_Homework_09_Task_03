Створіть проект за допомогою IntelliJ IDEA. 
Створіть клас MyDictionary. 
Реалізуйте у найпростішому наближенні можливість використання його екземпляра. 
Мінімально необхідний інтерфейс взаємодії з екземпляром повинен включати:
1) метод додавання пар елементів,
2) індексатор для отримання значення елемента за вказаним індексом
3) і властивість тільки для читання для отримання загальної кількості пар елементів.
